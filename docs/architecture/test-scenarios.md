## F-01. 시스템 설치

| 구분 | 케이스 | 조건 | 기대 결과 |
|------|--------|------|-----------|
| 정상 | 최초 설치 성공 | 유효한 로그 파일 경로, Dify 정상 응답 | `config/setup.properties` 생성, 4개 설정값 저장 |
| 정상 | 재설치 | `config/setup.properties` 이미 존재 | 기존 파일 덮어쓰고 재저장 |
| 예외 | 사전 조건 위반 — 파일 없음 | 존재하지 않는 경로 입력 | 설치 중단, 오류 메시지 출력 |
| 예외 | 사전 조건 위반 — 읽기 권한 없음 | 권한 없는 파일 경로 입력 | 설치 중단, 오류 메시지 출력 |
| 예외 | 입력 오류 — 로그 파일 비어있음 | 0바이트 파일 지정 | 설치 중단 |
| 예외 | Dify 날짜 형식 추론 실패 | Dify API 오류 또는 타임아웃 | 재시도 N회 후 설치 중단 |

## F-02. 실시간 장애 감지

| 구분 | 케이스 | 조건 | 기대 결과 |
|------|--------|------|-----------|
| 정상 | 장애 없음 | 해당 구간 로그 있음, Dify가 장애 없다고 판단 | 알림 없음, 정상 종료 |
| 정상 | 장애 감지 | 해당 구간 로그 있음, Dify가 장애 감지 | Dify MCP → Swing 알림 발송 |
| 예외 | 사전 조건 위반 — 설정 파일 없음 | `config/setup.properties` 미존재 (F-01 미완료) | 배치 중단, 알림 |
| 예외 | 제약 — 해당 구간 로그 0건 | 읽기 구간 내 로그 없음 | Dify 호출 스킵, 정상 종료 |
| 예외 | Dify API 실패 | 호출 오류 또는 타임아웃 | 재시도 N회 후 배치 중단 |

## F-03. 시간 단위 이상 패턴 분석

| 구분 | 케이스 | 조건 | 기대 결과 |
|------|--------|------|-----------|
| 정상 | 분석 및 저장 성공 | 1시간 로그 있음, Dify 응답 1MB 이하 | `output/hourly/anomaly/yyyy-MM-dd_HH.dat` 저장 |
| 예외 | 사전 조건 위반 — 설정 파일 없음 | `config/setup.properties` 미존재 | 배치 중단, 알림 |
| 예외 | 입력 — 1시간 로그 0건 | 해당 구간 로그 없음 | Dify 호출 스킵, 파일 미생성 |
| 예외 | 제약 위반 — 응답 1MB 초과 | Dify 응답 content 크기 초과 | 배치 중단, 알림 |
| 예외 | Dify API 실패 | 호출 오류 또는 타임아웃 | 재시도 N회 후 배치 중단 |
| 예외 | 출력 실패 — 파일 저장 불가 | 디스크 권한 또는 용량 문제 | 배치 중단, 알림 |

## F-07. Hourly 파일 정리
