# CLAUDE.md

이 파일은 Claude Code가 이 프로젝트에서 작업할 때 참고하는 가이드 문서입니다.

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | LogAnalyzer |
| 설명 | 서버 로그를 주기적으로 수집하여 Dify 기반 RAG 파이프라인으로 이상 패턴을 분석하는 시스템 |
| 주요 기술 스택 | Java, Spring Boot, Dify Workflow, (DB 종류), AWS EC2 |
| 배치 주기 | 1분 / 1시간 / 1일 / 1월 단위 |

## 폴더 구조

```
프로젝트루트/
├── CLAUDE.md                  ← 현재 파일
├── docs/
│   ├── architecture/          ← 시스템 구조 다이어그램 (컴포넌트, 데이터 흐름)
│   ├── design/                ← 함수/메서드 단위 상세 설계서
│   └── guidelines/            ← 코드 스타일, 언어 규칙, 개발 가이드라인
├── src/
│   └── ...
└── ...
```

## 문서 참고

작업 전에 아래 문서들을 우선 확인해 주세요.

- **시스템 아키텍처**: `docs/architecture/` — 기능 정의, 컴포넌트 구성, 데이터 흐름 등 (`docs/architecture/index.md` 참고)
- **개발 가이드라인**: `docs/guidelines/` — 코드 스타일, 언어 규칙, 네이밍 컨벤션 등
  - `tech-stack.md` — 기술 스택 및 배치 구현 방식 (배치 방식 미정)
- **함수 설계문서**: `docs/design/` — 함수 설명 및 구체적인 함수 설계 (`docs/design/index.md` 참고)
